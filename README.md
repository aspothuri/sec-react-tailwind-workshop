# Quickstart
1. Clone the repository to your machine.
```bash
git clone https://github.com/aspothuri/sec-react-tailwind-workshop.git
cd sec-react-tailwind-workshop
```
2. Install all dependencies.
```bash
npm install
```
3. Run app in development mode.
```bash
npm start
```
